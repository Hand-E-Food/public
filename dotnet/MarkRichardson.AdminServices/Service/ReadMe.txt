# How to Use


1.	Add all method signatures to `IAdminServices.cs`
2.	Implement all methods in `AdminsServicesService.cs`
3.	Name the service:
	a.	Open `ProjectInstaller.cs` in the designer.
	b.	Select the `serviceInstaller` object.
	c.	Change the properties for:
		*	ServiceName
		*	DisplayName
		*	Description
